# RiyaBot
ちょっとエッチなbot

## コマンド
  ```bot関係```
  
  `help` - ヘルプを表示します
  
  `ping` - pingを返します
  
  
   ```image関係```
   
   `ass` - assの画像を取得します
   
   `ecchi` - ecchiの画像を取得します
   
   `ero` - eroの画像を取得します
   
   `hentai` - hentaiの画像を取得します
   
   `maid` - maidの画像を取得します
   
   `milf` - milfの画像を取得します
   
   `uniform` - uniformの画像を取得します
   
   `oral` - oralの画像を取得します
   
   `paizuri` - paizuriの画像を取得します
   
   `selfies` - selfiesの画像を取得します
   
   `waifu` - waifuの画像を取得します
